## Comandos do git

Comando intrudutório
`` git int `` 


### Verificando informações

`` git status ``


### Realizando ações

adciona o arquivo a a arquivos estáqveis
`` git add 'nome-do-arquibo.a' ``
`` git add --ALL ``

realiza um commit
`` git commit -m "nome-do-commit" ``

## voltando para uma versão anterior

voltando para o ultimo commit
`` git checkout -- . ``
